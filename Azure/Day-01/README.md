# ☁️ Day 1: Cloud Computing & Azure Fundamentals

## 🚀 What is Cloud Computing?

Cloud computing is the delivery of computing resources (servers, storage, networking, software) over the internet on a **pay-as-you-go basis**.

Instead of owning and maintaining physical hardware, users can access resources on demand—similar to utilities like electricity or water.

---

## ❗ Problems Solved by Cloud Computing

Traditional on-premises infrastructure had several challenges:

* High upfront cost for hardware and setup
* Difficulty in predicting future demand
* Long provisioning time (weeks/months)
* Maintenance of power, cooling, and networking
* Regular hardware upgrades (every 3–5 years)

Cloud computing eliminates these issues by offering scalability, flexibility, and cost efficiency.

---

## ☁️ Types of Cloud

* **Public Cloud** – Services offered over the internet to multiple users
* **Private Cloud** – Dedicated infrastructure for a single organization
* **Hybrid Cloud** – Combination of public and private cloud

---

## ⚙️ Core Cloud Principles

* On-demand self-service
* Scalability and elasticity
* Resource pooling
* Measured service (pay-per-use)
* High availability

---

## 🧩 Cloud Service Models

* **IaaS (Infrastructure as a Service)** – Virtual machines, storage, networking
* **PaaS (Platform as a Service)** – Development platforms without managing infrastructure
* **SaaS (Software as a Service)** – Ready-to-use applications via internet

---

## 🌐 Why Choose Azure ?

* Cost-effective and scalable solutions
* Strong security and compliance
* Global presence with multiple data centers
* Integration with Microsoft tools and services

---
## 🏗️ Azure Resource Hierarchy

* Management Groups
* Subscriptions
* Resource Groups
* Resources

This structure helps organize and manage cloud resources efficiently.

---

## ✅ Key Takeaway

Cloud computing allows organizations to focus on innovation and business growth instead of managing infrastructure, making it a fundamental technology in modern IT.
